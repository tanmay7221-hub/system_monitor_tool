# system_monitor_tool
## 🚀 Features

- 📊 Real-time CPU & RAM monitoring
- 🧾 Logs system data into CSV file
- 📈 Analyzes average and maximum usage
- 📉 Visualizes data using graphs (Matplotlib)
- 🖥️ Simple CLI-based interface

---

## 🛠️ Tech Stack

- Python
- CSV File Handling
- psutil (System Monitoring)
- Matplotlib (Data Visualization)

---

## 📁 Project Structure


system_monitor/
│
├── main.py # Entry point (CLI menu)
├── logger.py # Logs system data
├── analyzer.py # Analyzes logged data
├── visualize.py # Graph visualization
├── logs.csv # Stores system logs
└── README.md


---

## ▶️ How to Run

1. Clone the repository:

git clone https://github.com/tanmay7221-hub/system_monitor_tool.git

cd system_monitor_tool


2. Install dependencies:

pip install psutil matplotlib


3. Run the project:

python main.py


---

## 📊 Example Workflow

- Select option `1` → Logs system data  
- Select option `2` → Shows analysis  
- Select option `3` → Displays graph
